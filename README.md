# ClearPath Clone

Pixel-perfect rebuild of the Framer "ClearPath" therapy/coaching template
(reference: https://clearpath-template.framer.website/).

Stack: Next.js 14 · TypeScript · Tailwind · Framer Motion · Lenis.

## Quick start
```bash
npm install
npm run dev   # http://localhost:3000
```

## What's in here
- **BUILD_SPEC.md** — full design brief: tokens, type scale, section-by-section copy, animation notes.
- **CLAUDE.md** — how to drive the rest of the build in Claude Code.
- `app/`, `components/`, `lib/` — working Next.js scaffold. 9 sections are fully built;
  the rest render with correct copy and `// TODO` polish notes.
- `public/images/README.md` — the portrait files to add.

Start with `CLAUDE.md`, then add images, then polish the TODO sections.
